# v0.3.16

## Summary

This release focuses on the maintained Vue UI at `/` and refines the system-settings experience without changing the underlying routing workflow.

- refined the `主题与外观` panel into a more compact, glass-style layout that stays aligned with the rest of the dashboard cards
- added persistent button-color customization alongside the existing font-color controls
- polished the overview interactions and layout details that were recently verified on real deployments

## Added

- added button color settings persistence for both light and dark themes
- added runtime loading for the new button color appearance settings so the active theme applies them immediately

## Changed

- compacted the `主题与外观` module so labels, controls, upload/history actions, and sliders align more consistently across desktop and narrow layouts
- improved several overview-card spacing details, including the metrics controls and anchored statistics popover presentation
- refreshed the built-in Vue assets embedded in the binary for the current dashboard behavior

## Config impact

This release does **not** require any YAML changes.

If you are already on the current `config_all` / `config_up` package line, `v0.3.16` is a binary and WebUI update only.
