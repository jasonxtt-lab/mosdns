# v0.3.18

## Summary

This release fixes a remaining mobile-browser compatibility issue in the maintained Vue UI at `/`, specifically for the `最慢查询` card.

- fixed a cross-browser mobile layout bug where the `最慢查询` domain column could collapse entirely for some users after the previous narrow-screen adjustments
- replaced the fragile fixed-table-width calculation with a more compatible layout approach for that card

## Fixed

- changed the `最慢查询` table to use a safer column-sizing strategy on mobile browsers that do not reliably handle the previous `table-layout: fixed` plus `calc(...)` width combination
- kept the earlier single-line truncation behavior so long domain values still remain compact without collapsing the left column

## Config impact

This release does **not** require any YAML changes.

If you are already on the current `config_all` / `config_up` package line, `v0.3.18` is a binary and WebUI update only.
