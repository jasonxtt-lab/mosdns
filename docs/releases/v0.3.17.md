# v0.3.17

## Summary

This release fixes a mobile-layout regression in the maintained Vue UI at `/` and tightens a few related overview-card truncation rules.

- fixed the mobile `最慢查询` card so domain values no longer collapse into character-by-character vertical wrapping
- aligned the mobile overflow behavior of `Top 域名`, `Top 客户端`, and `分流统计` with safer single-line truncation

## Fixed

- reduced the mobile width pressure from the `最慢查询` latency column after the previous three-digit `ms` adjustment
- prevented overview-card primary text fields from inheriting aggressive emergency wrapping on narrow screens

## Config impact

This release does **not** require any YAML changes.

If you are already on the current `config_all` / `config_up` package line, `v0.3.17` is a binary and WebUI update only.
