# v0.3.13

## Summary

This release focuses on the maintained Vue dashboard at `/`:

- the overview card now provides anchored multi-window audit statistics with a compact hover panel
- audit retention wording is clearer and distinguishes hot detailed logs from time-window statistics
- the overview KPI and popover layout were refined for closer alignment with the live chart and better readability

## Added

- added `GET /api/v2/audit/stats/windows` for overview time-window statistics
- added built-in overview windows for:
  - `1小时内`
  - `最近6小时`
  - `24小时内`
  - `最近3天`
  - `最近7天`

## Changed

- the `/` overview trend card now supports a right-side anchored statistics panel instead of a plain tooltip
- the light-theme popover now uses a solid white background for clearer contrast and text readability
- overview KPI spacing and popover alignment were refined to better match the chart axis and title bounds
- system settings now describe audit capacity as a hot detailed-log limit rather than a generic long-term log capacity

## Fixed

- fixed missing test helper coverage for `pkg/upstream/transport/reuse_test.go`
- fixed popover theme styling so teleported panel colors follow the active light/dark theme correctly
- fixed compact-window overflow in the overview statistics panel

## Config impact

This release does **not** require manual YAML migration.
Existing deployments can upgrade in place.
