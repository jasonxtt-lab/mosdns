# v0.3.15

## Summary

This release fixes the real runtime behavior of `IPv4优先` and ships a round of verified dashboard polish for the maintained Vue UI at `/`.

- fixed `prefer_ipv4` so domains with valid `A` answers consistently suppress `AAAA` replies across direct, proxy, fakeip, and remembered-routing paths
- fixed the dual-selector execution path so internal reference queries no longer fail when a child sequence exits normally with a response
- refined the dashboard overview so ranking cards adapt to taller windows, the trend controls align better, and several list/status visuals are easier to scan

## Fixed

- fixed a `dual_selector` execution bug where shared recursive-walker state could cause `IPv4优先` to miss some `AAAA` suppressions
- treated `sequence.ErrExit` with an existing response as a normal completion inside `prefer_ipv4` reference queries
- added regression coverage for the `prefer_ipv4` path so delayed preferred-type lookups still block the non-preferred answer correctly

## Changed

- the overview page now expands the default visible rows of `Top 域名`, `Top 客户端`, `最慢查询`, and `分流统计` based on available window height, while keeping a minimum compact layout
- adjusted the overview trend card controls and footer presentation for cleaner alignment
- added several small dashboard readability cues, including unsaved markers, simple status dots, tab badges, and upstream latency color hints

## Config impact

This release does **not** require any new YAML changes beyond the config package already introduced for `v0.3.14`.

If you already updated to the `v0.3.14` config package, `v0.3.15` is primarily a binary and WebUI update.
